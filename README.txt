Acerca de:
---------------------

Esta instalación se realizo con el objetivo de consumir mostrar una información recibida por medio
de una URL para luego aplicar unos filtro sobre ella y mostrar un slider de imagenes.

La configuración base de la funcionalidad esta en el modulo indexcol dentro de la carpeta /modules/custom

Su respectiva base de datos esta en el archivo /indexcol_test.sql
